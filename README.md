# MiniGameDemo1
玩法介绍：
玩家操作游戏人物躲避机关收集关卡中的草药，学习草药的介绍，在最后关卡中利用学习的知识完成对草药的分类并合成医药古书。

游戏介绍：
unity引擎制作的平台跳跃类游戏。
与美术团队协作独立开发的MiniGame，游戏实现了对人物，音效，动画，UI系统的控制
使用射线检测实现了人物的多段跳，使用状态机实现了敌人AI，使用工厂模式和MVC架构制作了背包系统
对象池技术实现了持续机关，使用粒子系统制作了雨天特效，UGUI制作了游戏界面，背包ui
并将美术资源打包成图集提高渲染性能。
